# AI Diagnostic Expert System

This repository contains the deployment code for an intelligent medical diagnostic simulation, developed as the capstone mini project for the Knowledge Representation and Reasoning (KRR) Course. 

The system analyzes input symptoms against a professional Knowledge Base to provide a ranked list of potential diagnoses. Instead of relying on rigid binary logic, it utilizes a confidence weighted forward chaining engine to calculate probability matches, mirroring real world diagnostic nuances.

### System Architecture
The application is built entirely in Python and deployed via Streamlit. The architecture separates the knowledge base from the inference engine using Object Oriented Programming principles. 

Key technical features include:
* Forward Chaining Inference
* Confidence Weighted Subset Analysis
* Dynamic Vocabulary Parsing
* Matrix Diagnosis Ranking
